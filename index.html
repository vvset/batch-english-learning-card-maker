Batch English Learning Card Maker

A Codex and Claude compatible Skill package for batch-generating children's English learning cards, lesson pages, parent guides, review plans, quality reports, and downloadable ZIP packs.

中文 README English README 安装说明 Install

支持平台

  • Codex：复制到 Codex skills 目录即可使用。
  • Claude / Claude Code：支持自定义 Skills 时可导入同一个文件夹。
  • 也可以不安装，直接运行 PowerShell 脚本生成学习包。

它能做什么

  • 批量生成英语单词卡、短句卡、组合卡和多句学习页。
  • 每条内容包含英文、IPA 音标和中文意思。
  • 自动输出今日学习单、互动学习页、家长进度页和复习安排。
  • 生成家长使用包和完整检查包。

适合谁

  • 中文家长带孩子英语启蒙。
  • 老师批量制作课堂卡片。
  • 儿童英语资料制作者制作免费学习包。

入口文件

  • download.html：学习包总入口。
  • parent-use-pack.zip：发给家长的使用包。
  • qa-dashboard.html：制作者质检入口。

Platform support

  • Codex: copy the folder into your Codex skills directory.
  • Claude / Claude Code: import the same folder when custom Skills are enabled.
  • Direct script usage is also supported without installing the Skill.

What it does

  • Batch-generates vocabulary, sentence, combo, and lesson-page cards.
  • Includes English text, IPA phonetics, and Chinese meanings.
  • Creates a today learning sheet, interactive player, parent dashboard, and review plan.
  • Exports a parent-use ZIP and a full inspection ZIP.

Who it is for

  • Chinese-speaking parents teaching children English.
  • Teachers making classroom learning cards.
  • Creators sharing free English learning packs.

Main files

  • download.html: main learning-pack entry.
  • parent-use-pack.zip: parent-facing pack.
  • qa-dashboard.html: creator quality-check entry.
